### 网关接口前缀

为了方便网关的路由，考虑项目中的uri前缀将采用一下规则来控制

1. `/user` - > 用户
2. `/goods` - > 商品
3. `/cart` - > 购物车
4. `/comment` - > 评价
5. `/order` - > 订单
6. `/store` - > 商铺
